# BusNovaTech

## Descripción
BusNovaTech es un sistema desarrollado en Java como proyecto final del curso Estructura de Datos (CS-304). Permite la gestión de buses, usuarios y tiquetes, utilizando estructuras de datos como listas enlazadas y colas de prioridad.

## Objetivo
Implementar un sistema que simule la gestión de transporte, aplicando estructuras de datos para organizar y procesar la información de manera eficiente.

## Funcionalidades
- Registro de buses
- Gestión de usuarios
- Generación y control de tiquetes
- Atención mediante colas de prioridad
- Almacenamiento de datos en archivos JSON

## Tecnologías utilizadas
- Java
- Estructuras de Datos (listas, nodos, colas)
- JSON para persistencia de datos
- Maven

## Ejecución
1. Descargar el proyecto desde GitHub
2. Abrir en NetBeans o IntelliJ
3. Ejecutar la clase principal `ProyectoAvance1`

## Estructura del proyecto
- Clases de modelo: Bus, Ticket, Usuario
- Estructuras: listas enlazadas y colas
- Repositorios: manejo de datos en JSON
